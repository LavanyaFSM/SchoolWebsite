## ğŸ–¼ï¸ Adding Your Images

The website requires the following images. Please add them to the `images/` folder:

### Required Images:

1. **hero-bg.jpg** (1920x1080px recommended)
   - Hero section background
   - Should show: Happy children in a bright, colorful Montessori classroom

2. **about.jpg** (800x600px recommended)
   - About section image
   - Should show: Children working with Montessori materials, teacher guidance

3. **practical-life.jpg** (800x500px recommended)
   - Learning pathway image
   - Should show: Children doing practical life activities (pouring, sorting, etc.)

4. **sensorial.jpg** (800x500px recommended)
   - Learning pathway image
   - Should show: Montessori sensorial materials (pink tower, color tablets, etc.)

5. **language.jpg** (800x500px recommended)
   - Learning pathway image
   - Should show: Language learning activities, reading, storytelling

6. **community-1.jpg to community-6.jpg** (600x400px recommended)
   - Community gallery images
   - Should show: Various school activities, outdoor play, classroom scenes

7. **program-1.jpg to program-6.jpg** (700x500px recommended)
   - Program cards
   - Should show: Different program activities (nature, arts, math, culture, life skills, social)

### Image Guidelines:

- Use high-quality, bright, and colorful images
- Show diverse children engaged in activities
- Natural lighting works best
- Avoid stock photos that look too staged
- Images should reflect the Montessori philosophy

### Free Image Resources:

- **Unsplash**: https://unsplash.com/ (search: "kindergarten", "montessori", "children learning")
- **Pexels**: https://www.pexels.com/ (search: "preschool", "kids classroom")
- **Pixabay**: https://pixabay.com/ (search: "children education")

## ğŸš€ How to Use

1. **Add Your Images**: Place all required images in the `images/` folder
2. **Customize Content**: Edit `index.html` to update:
   - School name and contact information
   - Address and phone numbers
   - Program descriptions
   - Statistics (years, students, teachers)
3. **Adjust Colors** (Optional): Edit CSS variables in `styles.css`:
   ```css
   :root {
       --primary-color: #FF6B6B;    /* Main brand color */
       --secondary-color: #4ECDC4;   /* Secondary accent */
       --accent-color: #FFE66D;      /* Highlight color */
   }
   ```
4. **Open in Browser**: Simply open `index.html` in your web browser
